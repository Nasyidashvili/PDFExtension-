# PDF Extension 

Chrome Extension (Manifest V3) 

- Tab — მომდევნო წინადადება
- Shift + Tab — წინა წინადადება

## ჩატვირთვა Chrome-ში

1. 'chrome://extensions' საიტზე უნდა ჩავრთოთ 'developer mode'
2.  'Load Unpacked' დაჭერის შემდეგ ვირჩევთ ფოლდერს, სადაც manifes.json მდებარეობს


## ფუნქციონალის შემოწმება

1. გახსენი PDF ბმული Chrome-ში extension ავტომატურად გადაგამისამართებს custom viewer-ზე
2. დააჭირე 'Tab' გამოჩნდება ხაზგასმა პირველ წინადადებაზე, ყოველ დაჭერაზე შემდეგზე გადავა
3. დააჭირე 'Shift + Tab' ხაზგასმა წინა წინადადებაზე დაბრუნდება
